## 개발 환경
- java11
- Srping 5.0.7 
- MySQL
- MyBatis
- Git,github
- AWS EC2

## 구현기능

- 게시글 생성/ 읽기/갱신삭제

- 댓글 생성/읽기/갱신/삭제
